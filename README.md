# 2D Metroidvania: Modular Exploration Framework

## 🎯 Overview
This project is a 2D Metroidvania built in Unity, focusing on **Finite State Machine (FSM)** architecture and **Gameplay Balancing**.

## 🎥 Gameplay & Testing Demo
https://youtu.be/bepntIV-zMw

## 🛠 Key Responsibilities (QA & Balancing)
As the **QA Tester & Game Balancer**, I focused on:
- **FSM Testing:** Verified 10+ player states to ensure bug-free transitions.
- **Difficulty Tuning:** Balanced enemy AI stats (damage, speed, vision) for a fair progression.
- **Combat Feel:** Calibrated Input Buffering windows to enhance responsiveness.

## 📂 Project Highlights
- **Architecture:** Generic State Machine, Interface-based Combat (`IDamageable`).
- **Optimization:** Object Pooling for projectiles.
- **Systems:** Custom Scene Management and Data Persistence.
